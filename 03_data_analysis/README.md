# 📊 Data Analysis

En esta fase se realizaron análisis de negocio utilizando el modelo estrella construido previamente.

El objetivo es transformar datos en insights que apoyen la toma de decisiones.

---

## 🧠 KPIs Principales

Se calcularon métricas clave del negocio:

* **Total Revenue**: ingresos totales generados
* **Total Orders**: número de órdenes únicas
* **Average Order Value (AOV)**: valor promedio por orden

---

## 📈 Análisis Temporal

Se evaluó el comportamiento de ventas en el tiempo:

* Revenue mensual
* Crecimiento mes a mes (Month-over-Month)
* Identificación de meses con alto y bajo rendimiento

---

## 👥 Análisis de Clientes

Se analizaron los patrones de consumo:

* Top clientes por gasto total
* Segmentación de clientes:

  * VIP (alto gasto)
  * Frecuentes
  * Ocasionales

---

## 📦 Análisis de Productos

Se evaluó el desempeño de los productos:

* Productos más vendidos
* Revenue por categoría
* Identificación de categorías con mayor impacto en ingresos

---

## 🌎 Análisis Geográfico

Se analizaron las ventas por ubicación:

* Revenue por estado
* Identificación de regiones con mayor y menor desempeño

---

## 🔍 Insights Clave

* Existe una alta concentración de ingresos en un grupo reducido de clientes
* Se observan variaciones significativas en el crecimiento mensual
* Algunas categorías dominan el revenue total del negocio
* Existen regiones con bajo rendimiento que representan oportunidades de mejora

---

## 🎯 Conclusión

El análisis permite entender el comportamiento del negocio desde diferentes dimensiones (tiempo, clientes, productos y geografía), facilitando la toma de decisiones estratégicas.

---

## 🛠️ Tecnologías

* SQL Server
* Modelo Estrella (Star Schema)
* Funciones avanzadas (LAG, agregaciones)
