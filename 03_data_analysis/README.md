# 📊 Análisis de Datos

## 📌 Descripción
En esta etapa se realizaron consultas analíticas utilizando T-SQL para obtener métricas clave del negocio.

---

## 📈 KPIs Calculados

- **Ingresos Totales (Total Revenue)**
- **Total de Órdenes (Total Orders)**
- **Valor Promedio por Orden (AOV)**

---

## 🔍 Análisis Realizados

### 🔹 Tendencia de ingresos
Se analizó la evolución de ventas en el tiempo.

---

### 🔹 Clientes principales
Se identificaron los clientes con mayor contribución a los ingresos.

---

### 🔹 Productos más vendidos
Se determinaron los productos con mejor desempeño.

---

### 🔹 Análisis geográfico
Se evaluó la distribución de ingresos por estado.

---

## ⚙️ Técnicas Utilizadas

- Agregaciones (SUM, COUNT)
- GROUP BY
- JOINs
- Funciones de ventana (LAG)
- CTEs (Common Table Expressions)

---

## 🧠 Objetivo
Transformar los datos en información útil para la toma de decisiones.

---

## 🛠️ Herramientas
- SQL Server
- T-SQL
