# Data Modeling (Star Schema)

En esta fase se diseñó un modelo estrella para análisis de datos.

## Tablas creadas:

* dim_customer: información de clientes
* dim_product: información de productos
* dim_date: dimensión de tiempo
* fact_sales: métricas de ventas

## Características:

* Uso de GROUP BY para garantizar unicidad
* Definición de Primary Keys
* Relaciones mediante Foreign Keys

## Objetivo:

Optimizar los datos para análisis en herramientas BI.
